# 🚀 Aprendendo Spring Boot

Este repositório foi criado com o objetivo de documentar minha jornada de aprendizado em **Spring Boot**, abordando desde conceitos básicos até aplicações mais avançadas no desenvolvimento de APIs e sistemas backend em Java.

---

## 📚 Sobre o projeto

Aqui você encontrará exemplos práticos, anotações e pequenos projetos que desenvolvo enquanto estudo:

- Fundamentos do Spring Boot  
- Criação de APIs REST  
- Integração com banco de dados  
- Arquitetura em camadas  
- Boas práticas de desenvolvimento  
- Uso de ferramentas como Maven e Lombok  

---

## 🛠️ Tecnologias utilizadas

- Java  
- Spring Boot  
- Maven  
- JPA / Hibernate  
- Lombok  
- Banco de dados (H2, MySQL, etc.)
